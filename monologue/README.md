# 🎙️ Monologue AI

**Monologue** 

Landing page template made with Next.js, Tailwind CSS, GSAP, and Framer Motion.


## 🛠️ Tech Stack

This project is built with the latest industry-standard technologies for maximum performance and a premium user experience:

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations**: 
  - [GSAP](https://gsap.com/) (For professional text staggering and 3D reveals)
  - [Framer Motion](https://motion.dev/) (For smooth scroll-driven effects and UI transitions)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Optimized via `next/font` (Inter & Outfit)

---

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Development Server**:
   ```bash
   npm run dev
   ```

3. **Open the App**:
   Navigate to [http://localhost:3000](http://localhost:3000) to view the site.
